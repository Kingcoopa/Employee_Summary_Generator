# Employee_Summary_Generator
Node CLI that will generate profile for employees.

## Description
This app that uses Inquirer and Node to compile user data and render a HTML file that shows the team's info; ie emails, GitHub profiles, etc. This is done after the tests are completed.

## Usage
The app utilizes prompts which builds a team of a manager, intern and engineer. Once the user answers the prompts in the terminal. Then  a team.html is created in the output directory.

The app also makes utilizes validation to make the user inputs character values: 'name', 'id' and 'email' which prevents errors.
